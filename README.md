人犬牵绳检测系统

几年前写的前端页面了，样式不太行请见谅

基于YOLOv5和Flask的智能人犬牵绳检测系统，能够自动识别图像或视频中的人与狗，并检测是否存在牵绳行为。

项目概述

本项目使用YOLOv5目标检测模型，结合Flask框架构建了一个Web应用程序，可以检测图像或视频中的人与狗是否使用牵绳。系统能够识别:

    人(person)

    狗(dog)

    牵绳(leash)

功能特点

    🖼️ 支持图像上传和实时检测

    🎥 支持视频流处理

    ⚡ 高性能深度学习推理

    📊 可视化检测结果

环境要求

    Python 3.8+

    PyTorch 1.7+

    Flask 2.0+

    OpenCV
